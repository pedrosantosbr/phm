# Team photos

Drop member portraits here. They are referenced from `app/team/page.tsx`.

## Naming

- `m1.jpg` — first card (currently Pedro Santos)
- `m2.jpg` — second card (currently Mariana Matos)
- `m3.jpg` — third card (currently Fillipi Nascimento)

Order matches `team.members.m1/m2/m3` in `app/i18n/locales/{pt-PT,en-US}.ts`.

## Format

- Aspect ratio: **4:5 portrait** (e.g. 800×1000, 1200×1500)
- Format: `.jpg` (or change the extension in `app/team/page.tsx` if you prefer `.webp` / `.png`)
- Until a file exists, the page falls back to a placehold.co image automatically.
